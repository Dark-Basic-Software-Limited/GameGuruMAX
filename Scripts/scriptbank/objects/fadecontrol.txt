Fadecontrol Behavior

Fades the transparency of an object when player is in zone or switched by control.

Add a trigger zone or switch and logic link to the object to fade.
Attach this behavior to an object and set Physics ON/OFF, Always active ON.

Behavior Options:
Change PROMPT TEXT to suit.
Select the processing MODE to either Increase Visibility or Decrease Visibility
Change the LEVEL of fade transparency
Change the fading SPEED
<Sound0> to play a sound when fading

When faded to 0% object will be destroyed.
