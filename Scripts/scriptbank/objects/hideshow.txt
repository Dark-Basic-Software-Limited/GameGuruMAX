HideShow
Will Hide or Show an object activated from trigger zone or switch

Attach to the object(s) to be hidden/shown. 
Set  and link to trigger Zone or to a swtich

Set the behaviours: 
Change the prompt text for when activated
Change Object Mode to either Hide or Show or Fade Hide or Fade Show
(If Show is selected the object will be automatically hidden on game start)
Change the delay before object is Hidden/Shown if required
Change the fade speed as object is Hidden/Shown if required

Play a sound when activated.
If used with a switch will toggle between hidden or shown
