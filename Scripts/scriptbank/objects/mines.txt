Mines

Allows pickup and deployment of an anti-personnel mine device.
Physics=ON, Always Active=ON, IsImobile=ON, Explodable=ON

Weapon Name: Put the weapon name you wish to use
Pickup Range: the distance before you can pickup
Trigger Distance: How close an entity needs to get before the mine is triggered
Enemy Hear Distance: Distance range nearby enemies can hear the explosion.
Pickup Text: Prompt Text when picking up
Placing Text: Prompt Text when placing the mine
Arming Text: Prompt Text when mine is primed/armed
Arming Delay: Delay time to move away from armed device.
Harm Player: Option to be harmed by planted mines

Sound0: pickup sound
Sound1: primed/armed sound
