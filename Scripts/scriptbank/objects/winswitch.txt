Winswitch behavior

This object will be treated as a switch object to end the level.

Behavior options:
Edit the PROMPT TEXT eg: "to End Level"
Select the PLAYER LEVEL for the min player level to be able use this switch 
Select GoToLevelMode for 'Use Storyboard Logic" or 'Go to Specific Level' to control
whether the next level in the Storyboard, or another level is loaded after the switch is turned on.
Check the ResetStates to reset game state on level load if returning to a previous leve if required.
Play the audio <Sound0> when the object is switched ON by the player.
