Object Mover

This will move an object by a specified amount and can be used as a 
switch object for activating other objects or game elements. 
Set Physics=ON, Always Active = ON, IsImobile=ON

Behavior Options:
Customise the PROMPT TEXT eg:"E to move"
Set the USE RANGE to adjust the range
Set the MOVEMENT TYPE either 'Push', 'Pull', 'Slide-Left', 'Slide-Right', 
'Slide-Up', 'Slide-Down', 'Rotate X', 'Rotate Y', 'Rotate Z'
Set the MOVEMENT amount 
Set to USE AS SWITCH to 'No' or 'Yes' to use as a switch 
to activate other objcts
<Sound0> will loop when used
<Sound1> will play when finished
