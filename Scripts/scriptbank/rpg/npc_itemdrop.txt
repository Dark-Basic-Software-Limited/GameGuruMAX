NPC Item Drop

This behavior allows a named item to be carried and dropped by a named npc.
Add this behavior to any object and set to Always Active On

Choose an object for the named item: Set to Physics=ON, Gravity=ON
Enter the NPC's Name in the behavior in the NPC NAME field.
Enter the Item's Name in the behavior in the ITEM NAME field.
Thats it

The Named NPC will have the item until it is killed then it will be
dropped for pickup as normal.
