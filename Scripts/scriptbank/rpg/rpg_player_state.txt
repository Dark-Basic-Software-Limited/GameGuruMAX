RPG Player State

A global behaviour for object handling user globals to incrementally 
add or deduct the players globals values  over time. This could be used
for Poisoning, Radiation, Hunger, Thirst etc.

Attach to an object, Set always active On.

Set the behaviours: 
Set the USER GLOBAL AFFECTED enter tha name of your User Global 
Set the AMOUNT of units to adjust
Set the TIME INTERVAL in seconds for adjustment
Set the EFFECT to either Add or Deduct
Set the DAMAGE LEVEL units above/below the AMOUNT to attribute damage to the health
Set the DAMAGE AMOUNT units of damage to health

Repeat this procedure for up to 4 User Globals you wish to affect by time.

Set the ActivateAtStart on/off - if unchecked use a switch or zone to activate this behavior.
