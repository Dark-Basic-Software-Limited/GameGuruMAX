Phantom Behavior

Ideally suited for the horror genre but could be used elsewhere.

The Phantom Behavior fades in an object/character and it can can approach player
when in zone or switched by control.
Attach to object and set Physics ON/OFF, Always active ON.
Then logic link to a switch or zone.

Behavior settings:
Change the PROMPT TEXT
Set the APPEARANCE MODE options are 'Show-Fade/Disappear', 'Show-Approach', 'Show-Face-Approach' , 'Show-Move-Fade/Disappear', 'Character Move-Once-No Collision'
Set the APPEARANCE LEVEL for how transparent the object/character is
Set the APPEARANCE AURA for the emmissive color glow
Set the APPEARANCE SPEED for the speed of appearing
Set the APPEARANCE TIME for the length of time befor of disappearing
Set the DISAPPEARANCE MODE for type of exit; Fade or Disappear
Set the MOVEMENT SPEED for how fast the object/character moves before dissapearing
Set the IDLE ANIMATION name for the entity if any
Set the MOVE ANIMATION name for the entity if any
Set the MOVEMENT X distance for movement
Set the MOVEMENT Z distance for movement
Set the <Sound0> for upon apperance
Set the <Sound1> for movement.

Adjusting the Appearence Level and Appearance Aura can make very effective ghostly apparitions.
