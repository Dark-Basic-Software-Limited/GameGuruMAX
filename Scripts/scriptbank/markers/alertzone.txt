Alert Zone

Attach to a trigger zone
The player may alert nearby enemies within the selected range when entering this Zone.

Behavior options:
Modify the PROMPT TEXT when entering the zone
Set the ALERT RANGE for enemies to hear
Set the ALERT MODE for single or multi use zone
Set the ZONEHEIGHT for the zones height
Set the SpawnAtStart - if unchecked use a switch or other trigger to spawn this zone
<Sound0> - Plays when entering the zone
