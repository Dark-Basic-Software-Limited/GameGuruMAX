Game Clock Behaviour

A global game clock for use by other scripts or for in-game. Set ALWAYS ON

Behavior settings:
You can DISPLAY CLOCK either Yes or No
Set the DISPLAY TYPE to either 24 Hour or 12 Hours display
Set the DISPLAY X screen position
Set the DISPLAY Y screen position
Set the DISPLAY TEXT SIZE
Set the START HOUR for the clock
Set the START MINUTES for the clock
Set the START SECONDS for the clock

Other scripts can check these globals for use
g_clock_seconds
g_clock_minutes
g_clock_hours
g_clock_althours
g_clock_timeofday
